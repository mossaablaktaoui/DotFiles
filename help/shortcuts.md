## Move between desktops or workspaces in gnome:
- Super + h/j/k/l

## Move between panes in wezterm:
- Alt + h/j/k/l

## resize a pane in wezterm:
- Alt + Shift + h/j/k/l

## Create panes in wezterm:
- Alt + \ for vertical panes.
- Alt + / for horizontal panes.
- I won't use panes(buffers) of nvim.

## Use System Scrolling (I can scroll everywhere):
- Super + j/k

## tilingshell extension shortcut:
- Super + Shift + h/j/k/l

## Lock screen:
- Super + Alt + l

## Move window one workspace:
- Super + Shift + Ctrl + h/l

## Chrome:
- Alt + t to create new tab.
- Alt + h/l move between tabs in chrome.

## Open WezTerm:
- Super + w

## Close a window:
- Super + e (may be q is better)

## Close a tab in Chrome:
- Alt + e (may be q is better)

## Close a pane in WezTerm:
- Alt + e (may be q is better)

## Move between workspaces:
- Super + 7/8/9/0 from the first workspace to the fourth.

## To Copy and Paste:
- Alt + y to copy.
- Alt + p to paste.

## Enable Copy mode in WezTerm:
- Alt + c

## Exit insert mode in WezTerm or nvim:
- jj

## To move between windows in nvim (like tabs in chrome):
- Shift h/l
